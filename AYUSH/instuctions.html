'clear' is used for clearing the screen of calculator

For calculating power of any number you should use Math.pow(a,b)

Example (1)- Math.pow(2,5) result will be 32

Example (2)- Math.pow(3,4) result will be 81

For calculating the value of Ï€(pi) you should use Math.PI

Example (1)- Math.PI result will be 3.14.

For calculating square root of you should use Math.sqrt(a)

Example: Math.sqrt(36) the result will be 6.

For calculating the value of trignometric functions like sin,cos,tan,etc.you should use Math.Trignometric function(Math.PI)

Example: Math.sin(Math.PI/2) result will be 1

For calculating round of any number you should use Math.round(a)

For calculating maximum between two or more numbers you should use Math.max(a,b)

Example: Math.max(2,3) result will be 3

Example: Math.max(2,3,4,5,6) result will be 6

For calculating minimum between two or more numbers you should use Math.min(a,b)

Example: Math.min(2,3) result will be 2

Example: Math.min(2,3,4,5,6) result will be 2

For getting next integer you should use Math.ceil(a)

Example:Math.ceil(5.6) result will be 6

For getting integer just before that number you should use Math.floor(a)

Example:Math.floor(5.6) result will be 5

For getting the absolute value of any number you should use Math.abs(a)

Example:Math.abs(-57) result will be 57
